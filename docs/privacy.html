Privacy Policy

Terakhir diperbarui: 26 Januari 2026

Ringkasan

ServisKu adalah aplikasi offline. Data servis kendaraan disimpan di perangkat pengguna. Kami tidak menjual data pengguna.

Data yang disimpan (lokal di perangkat)

Data yang kamu masukkan di aplikasi, misalnya: profil kendaraan (opsional plat), catatan servis (tanggal, km, item, biaya, catatan), dan lampiran (misalnya foto struk) disimpan secara lokal di perangkat.

Data yang dikumpulkan

ServisKu tidak mengumpulkan data pribadi ke server (tidak ada akun, tidak ada backend pada versi offline). Kami juga tidak menggunakan iklan, analytics, atau SDK pelacakan untuk mengumpulkan data pengguna.

Jika kamu membagikan file ekspor/backup melalui aplikasi lain (misalnya email/WhatsApp/Drive), pemrosesan selanjutnya mengikuti kebijakan privasi aplikasi pihak ketiga yang kamu pilih.

Izin perangkat

Beberapa fitur mungkin membutuhkan izin berikut (sesuai yang kamu aktifkan):

  • Notifikasi: untuk pengingat servis.
  • File/Foto (opsional): hanya saat kamu memilih lampiran (misalnya foto struk) melalui pemilih file/foto. ServisKu tidak mengakses file di latar belakang.

Ekspor & backup

Kamu dapat mengekspor data (misalnya PDF riwayat servis) atau melakukan backup secara manual. File ekspor/backup hanya dibuat ketika kamu memintanya.

Penghapusan data

Kamu dapat menghapus data dengan menghapus entri di aplikasi. Menghapus aplikasi (uninstall) juga akan menghapus data lokal dari perangkat.

Keamanan

Kami berupaya menjaga keamanan aplikasi. Namun, keamanan perangkat juga dipengaruhi oleh pengaturan perangkatmu (PIN, biometrik, dll).

Perubahan kebijakan

Jika di masa depan ServisKu menambahkan fitur online (misalnya sinkronisasi cloud), kebijakan ini akan diperbarui.

Kontak

English (Short)

ServisKu is an offline app. Your data is stored locally on your device. We do not collect personal data on servers (no account, no backend for offline version). We do not use ads, analytics, or tracking SDKs. Permissions may be used for notifications (service reminders) and optional attachments (picked by the user). If you share exported files via third-party apps, their privacy policies apply. Contact: karimazodaraalifiana@gmail.com.
